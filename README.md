## masterment instructions:

1. assume that `nodejs` is installed on your machine.
2. clone project:
    ```
      $ git clone git@github.com:pencilpix/snake.git
    ```
3. run project
    ```
      $ npm install
      $ npm start
    ```
4. to build
    ```
      $ npm run build
    ```
5. enjoy!
